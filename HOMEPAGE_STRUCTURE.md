# Полный разбор структуры главной страницы (HomePage)

**Файл:** `src/pages/HomePage.tsx`
**Обёртка страницы:** фон страницы `#0a0a0a` — почти чёрный
**Шрифт:** Manrope (переменный, вес 200–800), подключён из `/fonts/Manrope-VariableFont_wght.ttf`
**Основной цвет текста:** `#ffffff` (белый)
**Акцентный цвет:** `#f80000` (красный), при наведении: `#ff3333`

---

## Общие стили (используются во всех секциях)

| Класс | Что делает |
|---|---|
| `.card-dark` | Тёмная карточка: фон `#111111`, тонкая серая обводка `#222222`, скруглённые углы |
| `.card-hover` | При наведении карточка поднимается вверх на 8px и появляется тень снизу |
| `.btn-primary` | Красная кнопка `#f80000`, белый текст, жирный, буквы чуть разрежены. При наведении — светлее + красное свечение |
| `.glass` | Полупрозрачный тёмный фон с размытием фона за элементом |

## СЕКЦИЯ 1: HeroSection

**Файл:** `src/components/sections/Home/HeroSection.tsx`

### Размеры и фон
- **Высота:** 300% высоты экрана (скролл внутри секции)
- **Фон:** фоновое изображение `/HeroSent/sand-pile.png`
  - Масштаб фона: растянут на весь блок
  - Позиция фона: по центру сверху (на больших экранах — строго по центру)
  - Фон не повторяется
- Всё, что выходит за края — обрезается

### Контейнер контента
- **Отступы по бокам:** 1 см слева и справа
- **Отступ сверху:** 112px
- **Отступ снизу:** 64px
- **Минимальная высота:** 200% экрана
- **Расположение:** элементы в столбик сверху вниз
- **Слой:** поверх остального контента

### Заголовок (h1)
- **Позиция:** верхний левый угол
- **Максимальная ширина контейнера:** 448px
- **Анимация:** появляется снизу с затуханием (0.8 сек)
- **Размер текста:** 48px на телефоне → 60px → 72px → 96px на большом мониторе
- **Толщина шрифта:** тонкий (300); слово «Кварцевый» — средний (500)
- **Высота строки:** 0.95 (строки прижаты друг к другу)
- **Межбуквенный интервал:** сжатый (-0.025em)
- **Цвет:** белый

### Блок описания + кнопки
- **Позиция:** прикреплён к правому краю, снизу отступ 105% экрана
- **Отступы по бокам:** 1 см
- **Выравнивание:** прижат к правому краю
- **Текст описания:** максимальная ширина 448px, выровнен по правому краю
  - Цвет: серый (приглушённый)
  - Размер: 18px
  - Межстрочный интервал: просторный (1.625)
  - Отступ снизу: 32px
- **Кнопки:**
  - На телефоне — в столбик, на планшете+ — в строку, промежуток 16px
  - «В каталог»: красная кнопка, белый текст, скруглённые углы, жирный
  - «Связаться»: прозрачная с серой обводкой, белый текст, при наведении обводка становится красной

### Индикатор прокрутки
- **Позиция:** левый край, 90% от верха экрана (внизу видимой области)
- Текст серый, маленький, все заглавные, буквы разрежены
- Иконка стрелки вниз — анимация прыжка (bounce)

### Блок цитаты (посередине)
- **Позиция:** прикреплён к низу, отступ 50% экрана
- **Отступы:** 1 см по бокам, 100px сверху
- **Размер текста:** 36px → 48px → 60px
- **Толщина:** тонкий (300)
- **Высота строки:** 1.05
- **Цвет:** белый

### Карточки характеристик (6 штук)
- **Позиция:** прилеплены к низу секции (на больших экранах — с отступом 15% от низа)
- **Отступы по бокам:** 1 см
- **Расположение:** на телефоне — в столбик, на большом экране — в строку с промежутками
- **Левая группа (3 карточки):** выезжают слева при появлении, текст вправо
- **Правая группа (3 карточки):** выезжают справа при появлении, текст влево
- **Расстояние между карточками:** 8px на телефоне, 32px на большом экране
- **Каждая карточка:**
  - Тёмная полупрозрачная (`#111111`, 60% непрозрачности), лёгкое размытие фона
  - Внутренний отступ: 24px со всех сторон
  - При наведении: красноватая обводка
  - Иконка: 28×28px, белая
  - Подпись (label): маленькая, серая, заглавные буквы, разреженные
  - Значение: 24–30px, тонкий шрифт, белый
  - Описание: серый, 16px, просторный интервал

### Декоративный фон
- Радиальный градиент: круг 800×800px, тёмно-синий → прозрачный, по центру
- Сетка линий: синяя клетка 100×100px, прозрачность 2% (почти не видно)

| Порог | Ширина экрана | Что меняется |
